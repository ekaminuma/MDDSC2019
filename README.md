## 基礎編修了テスト

- MDDSCの基礎編の修了には、オンラインテストで満点を取る必要があります。
- Google Formのテスト機能を使って、オンラインでテストを実施できる様に構築しました。
- オンラインテストの条件設定
   + 何度でも投稿可能
   + テスト投稿時に、結果が登録したe-mail addressに送信される。
   + テスト結果である点数表示と共に「間違えた問題のヒント」が提示される。忠実に実行すれば、正解に辿り着ける程度のヒントを作成した。
